## Description
Slack webhook API client for Swift.

## Requirements
- Swift 2.1 or later
- iOS 8.0 or later

## Installation

#### [Carthage](https://github.com/Carthage/Carthage)

- Insert `github "to4iki/Monocle" ~> 0.0.3` to your Cartfile.
- Run `carthage update`.
- Link your app with `Monocle.framework` in `Carthage/Checkouts`.
